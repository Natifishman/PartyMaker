## 🛠️ Tools You'll Need

### Essential (Free):
- Android Studio Arctic Fox or newer
- Firebase Console access
- GitHub account for CI/CD
- LeakCanary (debug only)

### Recommended (Paid):
- Sentry.io account (~$26/month)

- BrowserStack for device testing (~$39/month)
